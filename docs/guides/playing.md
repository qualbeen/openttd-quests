# How to Play OpenTTD Quests

## Installation

1. Download the latest release from [GitHub Releases](https://github.com/qualbeen/openttd-quests/releases)
2. Extract the `openttd-quests` folder into your OpenTTD game scripts directory:
   - **Linux:** `~/.openttd/game/`
   - **macOS:** `~/Documents/OpenTTD/game/`
   - **Windows:** `Documents\OpenTTD\game\`
3. Start OpenTTD

## Starting a Game

1. Click **New Game**
2. Go to **AI/Game Script Settings**
3. Select **OpenTTD Quests** as the Game Script
4. Configure settings if desired, then start the game

**Important:** The Game Script must be selected *before* starting a new game.

## Playing

When the game starts, you'll only have access to basic buses and trucks. Open the **Story Book** (newspaper icon or hotkey) to see your quest log.

### Quest Log (Story Book)
- Shows available quests with descriptions and rewards
- Links to relevant map locations
- Updates as you progress

### Active Goals (Goal Window)
- Shows your current quest objectives with progress
- Objectives update automatically as you play

### Progression
- Complete quests to unlock new vehicle types and features
- Tier 0: Buses and trucks only
- Tier 1: Basic trains
- Tier 2: Ships
- Tier 3: Electric rail
- Tier 4: Aircraft
- Tier 5: Monorail
- Tier 6: Maglev

### Side Quests
- Optional quests generated from your specific map
- Offer cash bonuses and other rewards
- New side quests appear as you progress

## Co-op

In multiplayer, all players in the same company share quest progress. Any player's actions count toward quest objectives, and unlocks apply to the whole company.

## Compatibility

Works with any NewGRF vehicle set — the script dynamically classifies available engines into unlock tiers based on their properties.
